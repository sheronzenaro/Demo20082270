Project Contributors
This page lists all members who are part of the project.
How to add your name (GitHub fork & pull request):
1. Go to the original project repository on GitHub.
2. Click Fork to create your own copy of the repo.
3. In your fork, edit this file and update the contributors list in the JavaScript section.
4. Add your info as a new object:
{ name: "Your Name", github: "@yourUser" }
5. Commit the change to your fork.
6. Open a Pull Request to the original repository so the maintainer can approve your change.
